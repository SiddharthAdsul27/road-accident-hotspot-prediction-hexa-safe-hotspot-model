# 🚧 Road Accident Hotspot Prediction Model (HexaSafe)

## 📌 Overview

This project builds a **scalable geospatial risk ranking system** to identify and prioritize high-risk road accident zones using large-scale spatial and temporal data.

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* H3 (Hexagonal Geospatial Indexing)

## 🧠 Key Features

* 📍 **H3-based spatial aggregation** for location-aware modeling
* ⏱️ Temporal feature engineering pipeline
* ⚖️ Handles **severe class imbalance**
* 🚀 Optimized XGBoost model for ranking high-risk zones

## 📊 Results

* Precision@1–200 = **1.00**
* Precision@1000 = **0.979**
* Average Precision = **0.744**


## 📈 Use Case

* Smart city planning
* Traffic management
* Emergency response optimization
